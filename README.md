# kataract
Plot full-Stokes baseline-averaged dynamic spectra from Measurement Sets.

Usage:

```
$ python kataract.py 1647100873_sdp_l0_1024ch.ms
...
$ python kataplot.py 1647100873_sdp_l0_1024ch.ms_StokesI-kata.nc
```
